# APP_INCY
## Gastos Hormiga

# Descripción de la app 

Gastos Hormiga es una aplicación desarrollada en Flutter que permite registrar y visualizar pequeños gastos diarios, como café, comida, transporte u otros. La idea principal es ayudar al usuario a darse cuenta de esos gastos que normalmente pasan desapercibidos y que al final representan una cantidad importante de dinero.

# Funcionamiento de la aplicación

La aplicación funciona mediante el registro manual de gastos. El usuario puede ingresar la siguiente información:

- Monto del gasto  
- Categoría (café, comida, transporte, compras u otros)  
- Nota opcional  

Cada gasto se guarda en una lista dentro de la aplicación y se muestra en un historial para que el usuario pueda revisar lo que ha registrado.

Además, la aplicación calcula automáticamente el total gastado y muestra un mensaje sencillo dependiendo del comportamiento del usuario. Por ejemplo, si se detecta que el gasto en café es alto, se muestra una alerta indicando que ese tipo de gasto está aumentando.

La aplicación muestra el total acumulado de gastos, el mes actual y una lista con todos los gastos registrados. También incluye un mensaje de retroalimentación para ayudar al usuario a entender mejor sus hábitos de consumo.

Este proyecto se desarrolló como un prototipo funcional para demostrar cómo una aplicación sencilla puede ayudar a las personas a llevar un mejor control de sus gastos diarios sin necesidad de utilizar herramientas complejas.
